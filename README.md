# EngineCC - Máy Tính Dung Tích Xi-lanh

**EngineCC** là một ứng dụng Android mạnh mẽ, đơn giản và hiện đại giúp các kỹ thuật viên, sinh viên cơ khí hoặc những người yêu xe tính toán chính xác dung tích xi-lanh (cc) của động cơ đốt trong.

## 🚀 Tính năng nổi bật
- **Giao diện Material Design 3:** Trải nghiệm người dùng hiện đại, mượt mà và trực quan.
- **Tính toán nhanh:** Chỉ cần nhập các thông số kỹ thuật cơ bản.
- **Tối ưu hiển thị:** Bố cục gọn gàng, hiển thị tốt trên nhiều kích thước màn hình.
- **Hỗ trợ đa xi-lanh:** Tính toán cho động cơ từ 1 xi-lanh đến nhiều xi-lanh.
- **Phản hồi tức thì:** Hiển thị lỗi ngay trên ô nhập liệu nếu dữ liệu không hợp lệ.

## 📐 Công thức tính toán
Ứng dụng sử dụng công thức chuẩn kỹ thuật:

> **V = (π × D² / 4) × L × n / 1000**

Trong đó:
- **D**: Đường kính piston (mm).
- **L**: Hành trình piston (mm).
- **n**: Số lượng xi-lanh.
- **V**: Dung tích xi-lanh (cm³ hay còn gọi là cc).

## 🛠 Công nghệ sử dụng
- **Ngôn ngữ:** Java
- **UI Framework:** Android Material Components (Material 3)
- **Layout:** ConstraintLayout, CoordinatorLayout
- **Công cụ:** Android Studio

## 📥 Cài đặt
1. Clone dự án:
   ```bash
   git clone https://github.com/LumiSnow/EngineCC.git
   ```
2. Mở dự án bằng **Android Studio**.
3. Build và chạy trên thiết bị Android của bạn (Yêu cầu Android 10.0+ / API 29+).
